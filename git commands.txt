git commands :

$ git clone [url] 
	to clone the repository at the url to my PC

$ git log 
	to see all the commits
$ git log --stat
	see all the commit with more informations

$ git diff [adress old] [adress new]
	compare 2 commits 

$ git init
	create an empty git repository

$ git add [file name]
	add a file in the staging area
